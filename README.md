# NotsApp
- NotsApp is phone number and otp based chat Application.


                                     STACK
                                     ✹✹✹✹✹
                                     
                 FRONTEND : Angular  BACKEND : NODEJS(ExpressJS)
                 API CALL : HTTPS, Websocket, 
                 Image Upload : AWS S3 Bucket to upload images,
                 OTP : ASW SNS service to get OTP. 
                 DB : MongoDB
